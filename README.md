# flutter_meetup

Aplicación desarrollada en Flutter para organizar los eventos realizados por Globant

## - Funcionalidades
* Busqueda de eventos
* Creación de eventos
* Favoritos
* Dark mode

## - Tecnologias utilizadas
* Provider
* Firebase
* Firestore
* Google sign in
* Material Design

## - Repository Pattern

El proyecto utiliza el patrón repositorio con ayuda de provider.
